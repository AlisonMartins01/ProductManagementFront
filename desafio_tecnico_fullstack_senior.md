
# Desafio Técnico – Desenvolvedor(a) Sênior Full Stack (.NET + Angular)

## 🎯 Objetivo
Construir uma aplicação full stack simples para **gestão de produtos**, com CRUD completo e boas práticas de arquitetura, código e testes.

## 🧱 Escopo

### Backend (.NET 6+)
- API RESTful com CRUD
- EF Core (InMemory/SQLite)
- Clean Architecture (opcional)
- DTOs + AutoMapper
- Swagger
- Validações
- Testes Unitários

### Frontend (Angular 14+)
- Listagem, criação, edição e exclusão
- Formulários reativos
- RxJS
- Interceptors e guards
- Validações

## ⭐ Diferenciais
- Docker
- CI/CD (GitHub Actions)
- Paginação, filtros
- Tratamento global de erros
- Documentação arquitetural
